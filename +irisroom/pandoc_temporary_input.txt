 
Graphics functions
===================
 
* [`bottomlegend`](../grfun/bottomlegend.html) - Horizontal graph legend displayed at the bottom of the figure window.
* [`ftitle`](../grfun/ftitle.html) - Add title to figure window.
* [`highlight`](../grfun/highlight.html) - Highlight specified range or date range in a graph.
* [`hline`](../grfun/hline.html) - Add vertical line with text caption at the specified position.
* [`maxfigure`](../grfun/maxfigure.html) - Create graphics window maximised across the entire screen.
* [`movetobkg`](../grfun/movetobkg.html) - Move graphics objects to the background.
* [`movetosubplot`](../grfun/movetosubplot.html) - Move an existing axes object or legend to specified subplot position.
* [`plotcircle`](../grfun/plotcircle.html) - Draw a circle or disc.
* [`plotpp`](../grfun/plotpp.html) - Plot prior and/or posterior distributions and/or posterior mode.
* [`plotmat`](../grfun/plotmat.html) - Visualise 2D matrix.
* [`plotneigh`](../grfun/plotneigh.html) - Plot local behaviour of objective function after estimation.
* [`vline`](../grfun/vline.html) - Add vertical line with text caption at the specified position.
* [`zeroline`](../grfun/zeroline.html) - Add zero line if Y-axis limits include zero.
 
Getting on-line help on qreport functions
==========================================
 
    help grfun
    help grfun/function_name
